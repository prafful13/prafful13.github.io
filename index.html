Personal Projects

Projects

k3s-dev

Docs ↗ GitHub ↗

A CLI that bootstraps a full local k3s development environment on macOS in one command. Running k3s-dev init runs pre-flight checks, installs the Bitnami Sealed Secrets controller, creates namespaces, and provisions a Postgres instance (PVC + Secret + Deployment + NodePort Service) — all tracked in ~/.k3s-dev/state.json.

Solves the "works on my machine" bootstrapping problem for teams using k3s as a local production replica. Postgres passwords are auto-generated and stored in macOS Keychain; the resulting k8s Secret is consumed directly by any project's Deployment without the project owning the postgres lifecycle. Also manages macOS LaunchAgents for host-level background services.

k3s / Kubernetes Bitnami Sealed Secrets Python 3.13 Typer macOS Keychain Bazel PostgreSQL LaunchAgents

robinhood-pilot

Live ↗ GitHub ↗

An automated equity trading bot using Robinhood's Agentic MCP API, deployed on a local k3s cluster. Trades on a configurable watchlist with four selectable strategies (RSI mean-reversion, MACD crossover, Bollinger Bands, RSI+MACD combo). All strategy and risk parameters are editable live from a Streamlit dashboard without a restart.

Infrastructure is handled by k3s-dev — Postgres, Sealed Secrets controller, and namespace bootstrapping are delegated to the shared CLI rather than embedded in the project. The project owns only its application secrets (OAuth tokens) and app manifests. Full zero secrets in git posture: three-layer model from macOS Keychain through Sealed Secrets to pod env vars.

Robinhood Agentic MCP API k3s / Kubernetes Python 3.13 Streamlit PostgreSQL Bitnami Sealed Secrets Docker Bazel Plotly

contracts-analysis

GitHub ↗

A local options screener dashboard built on Streamlit. Two modes: an Income Screener that finds cash-secured puts and covered calls to sell (ranked by annualized return on collateral), and a Buy Screener that surfaces high-delta directional plays ranked by a composite liquidity score.

Data is fetched live from Yahoo Finance (yfinance). Greeks (delta, gamma, theta, vega) are computed via Black-Scholes (py_vollib). The Annual% column renders as a progress bar — width encodes return magnitude so the best contracts are scannable at a glance without reading numbers. Filters (DTE, delta range, OTM%, volume, open interest) live in a persistent sidebar backed by config.yaml, editable without a rebuild. Deployed on local k3s via k3s-dev at NodePort 30502.
